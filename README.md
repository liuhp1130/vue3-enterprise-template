# fe-engineering-template

一个面向中大型前端项目的工程化模板，  
基于 Vite + Vue3 + TypeScript，  
内置规范、性能优化与自动化能力。

---

## 🚀 技术栈

- Vite
- Vue 3
- TypeScript
- Pinia
- Vue Router
- ESLint + Prettier
- Husky + Commitlint

---

## 📦 功能特性

- ⚡ 极速开发体验（Vite + ESM）
- 🧩 模块化工程结构
- 🛡 完整代码规范与提交规范
- 🚀 多环境配置（dev / test / prod）
- 📈 构建性能优化（分包、懒加载）
- 🤖 自动化构建与部署支持

---

## 📁 项目结构说明

```txt
src/
├─ api        # 接口统一管理
├─ components # 通用组件
├─ hooks      # 自定义 hooks
├─ router     # 路由与权限控制
├─ store      # 全局状态管理
├─ utils      # 工具函数
├─ views      # 页面级组件
